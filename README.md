# Garden
<div >
    A semester project for the Computer Graphics course created in C++ using OpenGL and the PGR educational
    framework. The player takes on the role of a fairy whose task is to revive and beautify a plain garden.
    This is achieved by planting various types of flowers, grass, and placing stones. <br><br>
    <ul>
                <li><b>Interaction:</b> Precise planting and positioning of objects in 3D space using mouse clicks.</li>
                <li><b>Performance:</b> Smooth rendering of a large number of plants and stones without stuttering is
                  ensured by instanced rendering.</li>
                <li><b>Atmosphere:</b> The environment offers a dynamic day and night cycle, togglable fog, and a
                  flashlight for night exploration.</li>
                <li><b>Visuals:</b> The entire scene can be explored using four different camera modes and is brought to
                  life by an interactive flying bird.</li>
    </ul>
    <br>
    </div>
